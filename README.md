# Movie Recommendation System

## Project Overview

A hybrid recommendation engine developed using the MovieLens dataset. The project combines multiple recommendation approaches to improve recommendation quality and address limitations of individual models.

## Dataset

MovieLens Dataset

- 27,000+ Movies
- Millions of User Ratings
- Movie Metadata

## Project Workflow

1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis
3. Popularity-Based Recommendation
4. Content-Based Recommendation
5. Matrix Factorization (SVD)
6. Hybrid Recommendation Engine
7. Model Evaluation

## Models Implemented

### Popularity-Based Recommender
Ranks movies using average rating and rating count.

### Content-Based Recommender
Uses TF-IDF and cosine similarity on movie genres.

### Collaborative Filtering (SVD)
Predicts personalized ratings using matrix factorization.

### Hybrid Recommendation System
Combines popularity, content similarity, and collaborative filtering.
